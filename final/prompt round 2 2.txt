   Answer: C. Iron
   Explanation: The Eiffel Tower was constructed using iron, which was the primary material used by Gustave Eiffel for this iconic structure.

   Answer: B. Sequential
   Explanation: Functional, Object-oriented, and Procedural are well-known programming paradigms, whereas "Sequential" is not considered a distinct paradigm.

   Answer: D. Work function
   Explanation: The work function is the minimum energy needed to remove an electron from the surface of a material.

5. What is the purpose of an API (Application Programming Interface)?
   Answer: C. To allow different software systems to communicate with each other
   Explanation: APIs enable different software systems to communicate and interact with each other.

   Answer: D. Plotter
   Explanation: A plotter is an output device used for printing vector graphics, whereas scanners, cameras, and digitizers are input devices.

7. What is the main purpose of the virtual memory in an operating system?
   Answer: C. To provide more memory to applications than physically available
   Explanation: Virtual memory allows an operating system to compensate for physical memory shortages by temporarily transferring data from RAM to disk storage.

8. Which scientist is credited with the discovery of radioactivity?
   Answer: C. Marie Curie
   Explanation: Marie Curie is renowned for her research on radioactivity and won two Nobel Prizes for her work in physics and chemistry.

   Answer: C. Distillation
   Explanation: Distillation separates liquid mixtures by heating them to turn the component with the lower boiling point into vapor, which is then condensed back into liquid form.
